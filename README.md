## 🛠️ Projects & Scripts Showcase

### 1. Polygon Area Calculator
* **Description:** An object-oriented application that models Rectangles and Squares using class inheritance. It calculates areas, perimeters, diagonals, and generates a text-based visual representation of the shapes.
* **Concepts used:** OOP, Method Overriding, Class Inheritance, Polymorphism.
* **Directory:** `/01-polygon-area-calculator`

### 2. Budget App
* **Description:** A budget management system featuring expense/deposit tracking, category transfers, and an algorithm that generates a dynamic ASCII bar chart showing spending percentages by category.
* **Concepts used:** Object-Oriented Programming, Data Formatting, Data Visualization (Text-based).
* **Directory:** `/02-budget-app`

### 3. Tower of Hanoi Solver
* **Description:** A mathematical puzzle solver implemented via recursion that logs and tracks every state transition of the disks across the rods.
* **Concepts used:** Recursion, Stack data structure behavior, State logging.
* **Directory:** `/03-hanoi-tower-solver`

### 4. Custom Hash Table
* **Description:** A basic implementation of a Hash Table from scratch, showcasing custom string-to-integer hashing logic and handling potential key collisions via nested dictionaries.
* **Concepts used:** Hashing Functions, Data Structures, Lookups.
* **Directory:** `/04-hash-table`

### 5. User Settings Manager
* **Description:** A utility script that safely manages user configuration dictionaries with strict type checking, case-insensitivity handling, and input validation.
* **Concepts used:** Input Validation, Dictionary manipulation, String formatting.
* **Directory:** `/05-user-settings-manager`
